🧠Aplicación CRUD con Programación Asíncrona
📌Descripción

Este proyecto es una aplicación básica CRUD (Crear, Leer, Actualizar y Eliminar) que demuestra el uso de programación asíncrona utilizando Promesas y la sintaxis async/await. Proporciona una interfaz para gestionar datos, permitiendo agregar, eliminar y actualizar registros.

 Características

Interfaz de usuario para operaciones CRUD

Obtención de datos de forma asíncrona usando Promesas

Validación de los datos ingresados por el usuario

Almacenamiento y recuperación de datos usando localStorage

Exportar datos en formato JSON

Eliminar todos los datos

Tecnologías usadas

HTML5

CSS3 (con Bootstrap 5)

JavaScript (con Promesas y async/await)

Iconos de Font Awesome

Instalación

Para ejecutar este proyecto:

Clona el repositorio en tu máquina local

Abre el archivo index.html en tu navegador

Uso

Abre el archivo index.html en el navegador

Usa el menú de navegación para acceder a las secciones como Callback, Promesas y Async/Await

Completa el formulario para agregar datos y haz clic en "Agregar"

Usa el botón "Exportar JSON" para descargar los datos

Usa "Borrar todo" para eliminar todos los registros
